# Alias8n a.k.a Aliasolization

A simple tool to replace aliases in text files

* * *

## Installation 

    npm i alias8n 

## Using

Alias8n has several ways to replace aliases with its values. 

+ 
### Default
#### ctx.json
    {
        "hello": "Hello World!"
    }
#### index.html
    <p>a(:hello:)</p>
